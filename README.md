# Sales-Analysis
Project Description  This project focuses on analyzing sales data to identify key business insights such as revenue trends, product performance, and category-wise sales distribution. Using Tableau, an interactive dashboard is created to help stakeholders make data-driven decisions by visualizing sales patterns clearly and effectively.
The dataset used in this project was cleaned, processed, and analyzed to generate meaningful insights that can help improve sales strategy, inventory planning, and overall business performance.
